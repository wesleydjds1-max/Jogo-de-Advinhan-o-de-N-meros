# 🔢 Jogo de Adivinhação de Números em Python 🔢

Um jogo desenvolvido em Python onde o jogador deve descobrir um número secreto entre 1 e 100.

## Funcionalidades

- 🎮 Escolha da dificuldade
- 🔢 Número aleatório entre 1 e 100
- ✅ Validação de entrada
- ⚠️ Tratamento de erros
- 🏆 Sistema de pontuação
- 📈 Histórico das partidas
- 🥇 Recorde de melhor desempenho
- ⏱️ Cronômetro da partida

## Tecnologias

- Python 3

## Como executar

Clone o repositório:

```bash
git clone https://github.com/SEU-USUARIO/jogo-adivinhacao-python.git
```

Entre na pasta:

```bash
cd jogo-adivinhacao-python
```

Execute:

```bash
python Jogo de Advinhação de Números.py
```
